# Learning Journal - Style Guide

## Design Approach

I wanted a modern, clean look inspired by productivity apps like Notion, combined with Apple's glassmorphic design language. The goal was to create an educational tool that feels polished and motivating.

## Typography

**Font Families**: 
- Primary: Inter (clean sans-serif)
- Code: Fira Code (monospace for code snippets)

**Hierarchy**:
- Hero Headlines: 4xl-6xl, bold
- Section Headings: 2xl-3xl, semibold
- Body Text: base-lg, regular
- Labels: sm-xs, medium weight

## Colour Palette

**Primary Colours**:
- Purple (#7C3AED) - main accent
- Violet shades for gradients
- Slate for neutrals

**Theme Support**:
- Light mode: White backgrounds, subtle shadows
- Dark mode: Dark slate with glassmorphism effects

## Layout

**Spacing**:
- Small: 0.5rem
- Medium: 1rem
- Large: 1.5rem
- XL: 2rem

**Grid**:
- Desktop container: max-w-7xl
- Content regions: max-w-4xl
- Card grids: Responsive 1-3 columns

## Components

**Glassmorphic Cards**:
- Backdrop blur effect
- Semi-transparent backgrounds
- Subtle borders
- Rounded corners (rounded-2xl)

**Navigation**:
- Sticky header with blur
- Logo left, nav links centre
- Mobile hamburger menu

**Buttons**:
- Primary: Filled purple
- Secondary: Outline style
- Ghost: Transparent with hover

**Forms**:
- Clean input styling
- Floating labels
- Focus glow states

## Responsive Breakpoints

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Animations

Keep animations subtle:
- Card hover effects (slight scale, shadow)
- Progress bar fills
- Page transitions (fade)
